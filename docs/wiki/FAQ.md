# FAQ ❓

Ofte stillede spørgsmål om systemet.

**Q: Hvad er "Quantum Chatbot"?**
A: Det er en AI-agent (Gemini Pro) der er bygget direkte ind i sitet. Den kender mit CV og kan svare på spørgsmål som om den var mig.

**Q: Er dataen live?**
A: Ja! Alt data på Dashboard, Kort og Gæstebog hentes live fra en Supabase database.

**Q: Kan jeg bruge dette system selv?**
A: Ja, koden er open source. Se [Installation and Setup](Installation-and-Setup) for at komme i gang.
