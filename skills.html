Excel & VBA

Praktisk

Dashboards, modeller, automatisering og “clean” struktur.

  • Modeller: inputs → beregning → output → checks
  • Power Query / data-cleaning (hvis relevant)
  • VBA: import, format, loops, performance
Ressourcer

Økonometri & tidsrækker

Metode

Estimation, identifikation og robusthed – uden at “overfitte fortællingen”.

  • Regression, panel, robusthedstjek
  • Tidsrækker: forecasting, decomposition, diagnostic
  • Fortolkning + formidling

Makro & pengepolitik

Fagligt

Policy-relevant analyse: regler, transmission og heterogenitet.

  • Centralbanker, rente-/inflationsdynamik
  • Taylor-regler, empiriske rammer
  • Strukturerede notater og “policy-ready” output

Python

Data

Reproducerbare pipelines: data → analyse → figurer/tabeller.

  • Pandas/NumPy workflows
  • Plots, rapport-output, notebooks
  • Ryddelig struktur + dokumentation

SAS

Tidsrække

Klassisk, robust værktøj til tidsrækker og statistik (og ja, det virker stadig).

  • PROC ESM / UCM / VARMAX (hvis relevant)
  • Forecasting + model checks
  • Reproducerbart setup

GAMS / CGE

Modeller

Kalibrering, baseline-checks og stød i modelrammer.

  • SAM, kalibrering, closure
  • Shocks + resultattabeller
  • Konsistenschecks (Walras, baseline)
